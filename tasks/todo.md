# タスク：ゲストモードの廃止とローカルストレージ対応

## 背景
- Firebaseの匿名認証（ゲストモード）を廃止。
- 未ログイン時はブラウザのLocalStorageを利用し、ログインなしで即座に開始できるようにする。
- 後でGoogleアカウントでログインした場合、ローカルのデータをFirestoreへ同期・引き継ぎする。

## 完了定義
- [ ] ゲストモード（匿名認証）の開始ボタンが廃止されている
- [ ] アプリ起動時にログイン不要で即座に利用開始できる
- [ ] 未ログイン時にデータがLocalStorageに保存・取得される
- [ ] Googleログイン時にローカルのデータがFirestoreへ自動的に同期される
- [ ] 既存の機能（習慣、夢、目標、TODO、プロフィール、称号）がLocalStorageでも正常に動作する
